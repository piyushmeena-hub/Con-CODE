## 📝 Project Overview
**EduFlow** solves the twin problems of administrative inefficiency and student productivity. By replacing manual roll calls with **AI-powered facial recognition** and providing a **RAG-based tutor**, we turn wasted classroom time and "gap hours" into high-value learning sessions.

### 🚀 Key Features

<table>
  <tr>
    <td width="50%">
      <h4>🔍 Smart Attendance</h4>
      <ul>
        <li><b>Face Recognition:</b> Automated marking using OpenCV.</li>
        <li><b>Location Verification:</b> GPS-based geofencing to prevent proxies.</li>
        <li><b>Instant Alerts:</b> Email notifications sent to students immediately.</li>
      </ul>
    </td>
    <td width="50%">
      <h4>🤖 AI Study Assistant</h4>
      <ul>
        <li><b>RAG Integration:</b> Upload PDFs and chat with your notes.</li>
        <li><b>Context-Aware:</b> Summarizes long lectures and explains complex topics.</li>
        <li><b>24/7 Support:</b> On-demand academic help during free periods.</li>
      </ul>
    </td>
  </tr>
</table>

## 🛠️ Tech Stack

- **Backend & Frontend:** Python (Full Stack)
- **Database:** SQLite
- **Security:** Face Recognition + Geolocation Verification
- **AI Engine:** Retrieval-Augmented Generation (RAG) for document processing
